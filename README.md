# BOOKWORM Website - Final Project

A responsive website for book reviews and requests using vanilla HTML, CSS, and JavaScript.

 Features
- Home page with animated counters
- Reviews page with book listings
- Request form
- LocalStorage for saving data
- Responsive design for all devices

 Technologies Used
- HTML
- CSS
- Vanilla JavaScript
- LocalStorage

 How to Run
1. Download all files
2. Open `index.html` in any web browser
3. No installation or server needed
